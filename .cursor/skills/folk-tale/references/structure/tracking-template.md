# 长篇设定追踪模板

长篇创作时维护此文档，每完成 5 章回顾一次，保证地名、人名、超自然规则、时间线、伏笔与角色状态一致。

---

| 地名 | （如：青石镇、槐树沟） | | |
| 人名 | （如：陈三、王寡妇） | | |
| 建筑/地点 | （如：镇东祠堂、老井） | | |

*新增时补表，避免同人异名、同地异写。*

## 超自然规则清单

| 规则描述 | 适用对象/场景 | 例外或限制 |
|----------|----------------|------------|
| 例：鬼魂在鸡鸣前必须离开 | 所有亡魂 | 若被叫出真名则困住至天明 |
| （按实际设定填写） | | |
