# 寓言

**核心要素：** 寓意、教训、隐喻、世态人情、说教分寸

寓言是以小见大的民间叙事——借一个具体故事指向某种道理或世相，但好的寓言不说教，道理藏在故事里让听者自己品。与奇谈中的「荒诞寓言」相比：寓言更侧重**可提炼的教训或讽喻**，结构更简洁，寓意更明确；奇谈的荒诞寓言则偏**命运吊诡、世事讽刺**，结局常是「一声叹息」而非「一点道理」。

## 子类型

**教训型：** 贪心、懒惰、傲慢等招致后果——如人心不足蛇吞象、守株待兔。核心是「某种行为导致某种结果」，教训可被概括为一句话。
**讽喻型：** 借动物、精怪、奇事影射人情世态、官场市井。明写怪事，暗刺现实，不直接点破。
**警世型：** 触犯禁忌或违背天理遭报应。与怪谈的因果报应相近，但寓意更明确，常与民间禁忌、道德训诫绑定。
**智慧型：** 巧计、机智、以弱胜强。突出某个人物的聪明或某种处世智慧，留给听者「可以学」的意味。

## 寓言与奇谈的区别

| 维度 | 寓言 | 奇谈（荒诞寓言） |
|------|------|------------------|
| 寓意 | 可概括为一两句教训或讽喻 | 多为命运无常、世事吊诡，难以一句话说清 |
| 结构 | 相对短、线清晰，因果明确 | 可曲折多转，偏「你绝对想不到」 |
| 说书人 | 可在结尾轻轻点题，也可完全不点 | 多调侃、感慨，少直接教训 |
| 读者感受 | 「说的是这个理」 | 「这世道就这样」「命啊」 |

## 结构特点

- **篇幅：** 寓言多为短篇或超短篇，单线即可完成。若写中长篇，往往是多个小寓言串联（单元式）或一个主故事里嵌套若干寓言式片段。
- **因果链：** 寓言的因果要清楚——读者能回溯「因为这样，所以那样」。不追求奇谈式的意外反转，而追求「顺理成章后的那一下顿悟」。
- **角色：** 角色可类型化（贪婪者、老实人、精怪、动物），但需有一两个细节让人物立住，避免变成纯符号。

## 寓意与说教分寸

**宁可留白，不要喊口号。** 寓言最忌结尾变成「这个故事告诉我们……」或「所以做人不能贪心」。民间寓言的力量在于：故事讲完，道理在听者心里自己长出来。

**可用的分寸：**
- **完全留白：** 只讲故事，不说破。适合讽喻型、部分教训型。
- **说书人一句感慨：** 「这人啊，贪心不足。」「世上的事，多半如此。」——像点评，不是总结中心思想。
- **角色口中点题：** 由故事里的智者、老人、精怪说出一句可玩味的话，而不是作者跳出来说教。

**避免：** 叙述者直接归纳寓意、道德评判过于直白、结尾变成「所以我们应当……」

## 设定清单

- [ ] 核心寓意或讽喻指向什么？（一句话，但不必在文中写出）
- [ ] 类型偏教训 / 讽喻 / 警世 / 智慧？
- [ ] 有无超自然元素？（动物说话、精怪、报应等——寓言可有无均可）
- [ ] 说书人是否在结尾插话？若插话，语气是感慨还是点评？
- [ ] 时间与地域是否锚定？（寓言可虚化时代地点，也可落在具体乡土）

## 写作要点

**故事先行：** 先保证故事本身好看、可信、有细节，再让寓意自然浮现。若为了讲道理而硬编情节，寓言会塌。

**一个主寓意：** 一篇寓言通常只服务一个核心寓意或讽喻点。想说的太多，就拆成多篇或多个小节。

**语言俭省：** 寓言不必铺陈感官，可比怪谈、传说更俭省。把笔墨用在关键动作、关键对话和结局那一下。

**结尾的余味：** 结尾是寓言的灵魂。要么以一句说书人的轻叹收束，要么以一个画面、一个细节戛然而止，让听者自己品。

## 常见陷阱

- 结尾变成阅读理解答案 → 把寓意交给读者，叙述者不总结
- 人物完全符号化 → 给一两个具体细节，让人物像活人
- 为寓意扭曲情节逻辑 → 因果要在故事内部成立，不能为道理硬掰
- 说教腔 → 用说书人的口语、感慨代替作者的教训口吻
