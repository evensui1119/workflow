# 志怪（超自然记录体）

**核心要素：** 记录笔法、见闻辑录、简洁凝练、不加评判

志怪是中国最古老的超自然叙事传统——《搜神记》《述异记》《子不语》《阅微草堂笔记》。它的特点是"记录者"的姿态：不编故事，只记下听来的、见到的奇事，信不信由你。

## 与怪谈的区别

| 维度 | 怪谈 | 志怪 |
|------|------|------|
| 叙述者 | 故事的讲述者/亲历者 | 记录者/采集者 |
| 长度 | 可长可短 | 传统上偏短（几百到几千字），现代演绎可加长 |
| 目的 | 营造恐惧/感动 | 记录异事，"存而不论" |
| 修辞 | 浓烈、感官化 | 简洁、克制、笔记体 |
| 情感 | 有明确的情绪导向 | 冷静旁观，情感内蕴 |

**本技能中的志怪：** 取志怪的记录者姿态和简洁笔法，但在现代演绎中允许扩展篇幅、丰富感官、加入情节曲线。即"志怪的骨、故事的肉"。

## 志怪的叙事姿态

**记录者的克制：** "某县某人，夜行遇异"——不渲染，不议论，只记事实。恐惧和感动由读者自己产生。

**来源标注：** "余友某某言"、"里中老人云"、"旧志载"——标注故事的来源，增加可信度和距离感。

**存而不论：** 记录完毕不解释、不评价。偶尔加一句"此事甚异"已是最大的感叹。

**信息的节制：** 不多写一个字。人物只给必要信息（姓、籍贯、身份），场景只给关键细节。

## 设定要素

- [ ] 记录者是谁？（作者本人、友人转述、旧志摘录）
- [ ] 事件发生在何时何地？（用朝代/年号 + 地名锚定）
- [ ] 涉及何种异类/异象？（鬼、狐、物怪、异梦、奇病）
- [ ] 有无后续？（当事人后来怎样了？此事有无第二次发生？）
- [ ] 讲这个故事的意图？（存异、警世、怡情、无意图纯记录）

## 写作要点

**简洁是第一美德：** 传统志怪百字成篇。现代演绎虽可扩展，但仍应追求"少即是多"。每个句子都应有信息量。

**白描功夫：** 不用比喻堆积，用精准的动词和名词。"妇人发垂至踝"比"她的头发像瀑布一样长"更有志怪味道。

**"忽然"的力量：** 志怪最常用的转折词是"忽"——"忽有异声"、"忽不见"。平静叙述中突然出现异常，这个"忽"字就是全部的恐惧。

**结尾的断裂感：** 志怪故事常常在你最想知道后来怎样的时候戛然而止。"后不复见"——四个字就是结局。

## 现代演绎策略

将志怪的精神注入现代篇幅：

**短篇（1-2万字）：** 最接近传统志怪。可用"笔记集"形式——多个短小志怪串联，中间有记录者的串场。

**中篇（2-5万字）：** 取一个志怪条目展开。保留记录者框架，但内部故事铺展为完整的情节弧线。

**长篇（~10万字）：** 多条志怪线索交织。记录者在采集故事的过程中逐渐发现这些"独立的异事"之间有隐秘的联系。

## 结尾模式

- **戛然而止**："后不复见"、"此后无异"——干脆利落
- **旁证补充**："余后访其地，果有……"——记录者亲自验证
- **存疑结束**："此事甚异，姑记之"——不下结论
- **后续追踪**："后三年，又有人在该处……"——暗示还没完

## 常见陷阱

- 叙述太啰嗦 → 志怪的灵魂是简洁，砍掉一切修饰
- 解释太多 → "存而不论"是铁律
- 感情太外露 → 记录者应保持冷静，哪怕记的是极恐怖之事
- 丢失了"来源感" → 不交代故事从何而来，就失去了志怪的文体特征
