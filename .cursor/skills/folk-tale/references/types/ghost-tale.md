# 怪谈（鬼怪异闻）

**核心要素：** 鬼魂、妖怪、诅咒、阴阳交界、因果报应

怪谈的内核是"人与非人的遭遇"——不是简单的吓人，而是通过超自然的镜子折射人性。最好的怪谈让人毛骨悚然之后，留下悲凉或怅惘。

## 子类型

**鬼魂故事：** 死者归来——有冤报冤，有情还情。核心是"未了之事"（《聊斋·聂小倩》《牡丹灯笼》）
**妖怪故事：** 非人之物修炼成形，与人纠葛。核心是"异类的人性"（《白蛇传》《聊斋·婴宁》）
**诅咒/禁忌：** 触犯某种规则引发灾祸。核心是"不可触碰之物"（各地民间禁忌传说）
**阴间游历：** 活人误入冥界、地府、阴司。核心是"另一个世界的秩序"（《搜神记》中的冥界篇章）
**附身/夺舍：** 非人之物占据人体。核心是"你身边的人还是不是他"

## 情节引擎

怪谈故事的推动力通常来自以下几种：

1. **遭遇型**：主角无意中撞见超自然（走夜路遇鬼、借宿荒宅），被动卷入
2. **因果型**：主角或先人的行为触发了超自然报应（负心书生被鬼索命）
3. **禁忌型**：主角违反某种禁忌（打开不该打开的门、说了不该说的话）
4. **情缘型**：人与异类相恋（书生与狐女、猎户与花妖）

一个故事可以叠加多种引擎。

## 设定清单

- [ ] 超自然实体是什么？（鬼、妖、精怪、诅咒力量）
- [ ] 它为何出现？（冤屈、修炼、被召唤、自然存在）
- [ ] 它遵循什么规则？（怕阳光、怕公鸡叫、在特定时辰出现）
- [ ] 人物如何得知这些规则？（老人口述、亲身经历、翻旧书）
- [ ] 阴阳界限在哪里？（黄昏、坟地、老宅、水边、镜子）
- [ ] 因果逻辑是什么？（善恶报应 / 无因无果的纯粹恐惧 / 命运无常）

## 写作要点

**氛围先行：** 怪谈的恐惧60%来自氛围，30%来自想象，10%来自实际展示。在"遇鬼"之前，花充分的篇幅让读者感到"不对劲"。

**超自然的"日常化"：** 莫言式写法——鬼来了就像邻居来串门。越轻描淡写越可怕。不要用感叹号和形容词堆积来制造恐惧。

**人情味：** 最好的怪谈不只有恐惧。鬼有冤屈，妖有深情，精怪有孩子气。在恐怖的壳里装一个关于人（或非人）的故事。

**节奏模型：**
日常生活（建立温度）→ 第一个不对劲的细节 → 合理化（"大概是风吧"）→ 异常叠加 → 无法忽视 → 遭遇超自然 → 真相/高潮 → 余波（真的结束了吗？）

## 结尾模式

- **余音式**：故事讲完，叙述者补一句"后来再也没人见过她"——让读者自己想
- **轮回式**：同样的事在另一个时代/地方重演的暗示
- **因果落定**：善恶各有交代，但不必说教
- **开放留白**：最诡异的一个细节留在最后，不解释

## 常见陷阱

- 鬼一出场就全方位描写 → 延迟展示，用声音、气味、温度暗示
- 角色遇鬼后反应太理性 → 普通人的反应是惊、怕、僵、逃，不是分析
- 恐怖与温情的切换太生硬 → 用日常段落做过渡（吃饭、聊天、干活）
- 结尾解释太多 → 民间故事忌"科学解释"，留白是最大的恐惧
