# 莫言民间故事风格卡（深化版）

本风格卡在 novel-writer 技能中的莫言风格卡基础上深化，专门针对民间故事创作场景。

```
【风格卡：莫言 / 民间故事（深化版）】

叙述声音：
  人称：第一人称说书人为主
  框架：代际传承结构——"我爷爷说"、"老辈人讲"
  叙述者身份：既是听故事的人也是转述者，
              有时也是故事边缘的见证者
  距离：中远距离，用说书人的口吻保持一层纱，
        但在关键时刻可以突然拉近（"我爷爷说他当时浑身的汗毛都竖起来了"）
  可靠性：叙述者声称转述，暗示"不担保真假"——
          这种半信半疑正是民间叙事的黄金距离

句法节奏：
  句长：中长句为主体，铺陈绵延，逗号密集
  关键技法：
    - 逗号长句：不用句号打断叙述流，用逗号形成绵延不绝的讲述节奏
      一个段落可以只有一两个句号
    - 短句收束：长段铺陈后用一个短句兜住，形成节奏落差
      铺了一大段之后突然"完了"、"就这样"、"人没了"
    - 排比克制：偶尔用一组并列增势可以，但不连续排比铺陈，
      避免滑向朗诵腔或散文腔
  段落：偏长，一个段落可以涵盖一个完整的场景
  标点：逗号使用频率极高，省略号和感叹号极少

感官偏好：
  主导感官：五感并重，但嗅觉和触觉尤为突出
  密度：极高——一个场景可以铺四五层感官细节
  功能：让虚幻的事变得逼真，让超自然可以被"闻到"和"摸到"
  特色：
    - 嗅觉是莫言的招牌——腐烂的、发酵的、新鲜的、烧焦的、
      花香的、血腥的、汗味的……嗅觉世界极其丰富
    - 触觉偏粗粝：粗布、泥土、树皮、冰冷的铁、滚烫的炕
    - 色彩偏浓烈：红（血、高粱、嫁衣）、黑（土、夜、头发）、
      白（月光、骨、雪）
    - 味觉出现在食物和身体场景中：酒的辣、血的腥甜、
      泥土的涩、母乳的甜

意象修辞：
  来源域：乡土 / 身体 / 动物 / 食物 / 农事
  密度：浓烈，比喻密集但不显做作——因为取喻于日常生活
  核心意象群：
    - 土地意象：黑土、红土、黄土、泥浆、田埂
    - 食物意象：高粱、煎饼、豆腐、酒、母乳
    - 身体意象：汗、血、泪、骨、肉、乳房、肚子
    - 颜色意象：红（生命力/暴力/情欲）、黑（土地/夜/死亡）
    - 动物意象：狗（忠诚/卑微）、狐（诱惑/灵性）、
      驴（倔强/苦难）、蛙（喧嚣/生命力）
  修辞风格：
    - 追求新奇甚至粗野的生命力
    - 少用"像……"明喻和"不是……是……"句式，
      优先用暗喻或直接以具象细节替代比喻
    - 比喻偶尔出现时不求精致求震撼，把美的和丑的并置
    - 拒绝词藻堆砌：一个准确的动词胜过三个华丽的形容词
    - 通感可用但节制：声音有颜色、气味有形状，但不叠加使用

时间处理：
  基本线性，但嵌套——故事中的故事是主要的非线性手段
  关键场景极度膨胀：一个夜晚可以写一整章
  过渡快速压缩："过了年又过了年"一笔带过
  时间标记方式：节气、农事、民俗节日
    "麦子黄了那年"、"过了惊蛰"、"赶上三月三的庙会"
  季节感极强：每个季节都有独特的感官底色

对话风格：
  对话占比：偏少——叙述远多于对话
  出现时特征：
    - 口语化极强，农民说农民的话
    - 方言味浓但可读：取方言的神韵而非照搬方言
    - 标签简洁："说"、"道"、"嚷"，很少用复杂的对话标签
    - 对话常嵌在叙述中而非独立成段
  潜台词：不多——农民说话直来直去
  异类角色的对话：微妙地不同于凡人——
    略带古意，或语序微微不对，或用了人不常用的词

主题表达：
  方式：隐性为主——通过故事本身让读者感悟
  态度：悲悯中带戏谑，沉重中带狂欢
    不做道德判官，不下善恶结论
    对人性的弱点（贪、色、怯）持理解而非批判态度
    对苦难持"这就是命"的坦然
  结尾：开放留白，余韵悠长
    常以一个画面或一句淡淡的话收束
    不给答案，不做总结

标志性技法：
  1. 超自然日常化
     ——把不可能的事当作"本来就是这样"来写
     狐狸来敲门写得像邻居串门，死人说话写得像聊家常
     叙述者不惊讶，读者的惊讶因此加倍

  2. 感觉化叙述
     ——用气味、触感、色彩让读者"身临其境地相信"
     超自然之所以可信，是因为你能闻到那股味道、
     感觉到那阵寒意、看到那种不该有的红色

  3. 代际叙述框架
     ——说书人的代际传承赋予故事口耳相传的民间质感
     "我爷爷说"既是可信度的来源，也是距离感的控制器
     讲述行为本身成为故事的一部分

  4. 生命力的暴力美学
     ——不回避丑陋、粗鄙、血腥、肉欲
     把这些当作生命力的一部分来呈现
     美与丑并置，高尚与卑微并存——这就是民间

禁忌（莫言式民间故事绝不会做的事）：
  - 不用书面腔和知识分子式的抒情
    （不要"我凝望着月亮，想起了远方的故乡"）
  - 不解释超自然现象的因果
    （不说"因为她是狐狸精所以能变成人形"）
  - 不在结尾给出明确的道德判断
    （不说"这个故事告诉我们贪心没有好下场"）
  - 不用精致唯美的修辞
    （不说"她的眼睛如同秋水般清澈"，要说"她的眼睛亮得像刚宰的鸡眼"）
  - 不让叙述者全知全能
    （叙述者只知道他被告知的，其余的"谁知道呢"）
```

## 风格卡使用检查清单

每写完一个场景/章节，对照以下清单自检：

### 叙述声音
- [ ] 是在"讲"而不是"写"？
- [ ] 说书人的口吻是否一致？
- [ ] 代际叙述框架是否到位？

### 感官与意象
- [ ] 感官密度是否足够？（至少两种以上感官）
- [ ] 嗅觉和触觉是否有存在感？
- [ ] 比喻是否克制？有无"像……"明喻或"不是……是……"句式泛滥？
- [ ] 偶尔出现的比喻是否取自乡土/身体/动物？

### 语言节奏
- [ ] 逗号是否比句号多？
- [ ] 排比是否克制？（偶一为之可以，不连续使用）
- [ ] 铺陈之后是否有短句收束？
- [ ] 有无词藻堆砌、形容词叠加的段落？

### 超自然处理
- [ ] 超自然是否以平静口吻叙述？
- [ ] 是否避免了解释超自然的冲动？
- [ ] 超自然是否通过感官呈现而非概念描述？

### 禁忌检查
- [ ] 有无书面腔泄露？
- [ ] 有无知识分子式抒情？
- [ ] 有无道德判断的结尾？
- [ ] 有无过于精致的修辞？
