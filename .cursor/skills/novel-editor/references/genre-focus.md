# 类型评测侧重

不同小说类型的读者期待和核心竞争力不同，评分权重应相应调整。

## 类型专项评测要点

### 悬疑推理
- 诡计/谜面设计是否公平（线索对读者可见）
- 误导手法是否合理
- 逻辑推理链条是否严密
- 真相揭示是否有冲击力

### 言情/爱情
- 感情发展是否自然可信
- 情感冲突是否有真实基础（非强行误会）
- 角色互动是否有化学反应
- 甜/虐比例与节奏把控

### 奇幻/科幻
- 魔法/科技体系是否自洽
- 设定是否对情节有实质影响（非装饰性）
- 异世界/未来社会是否有说服力
- 奇观感与故事性的平衡

### 武侠/仙侠
- 力量体系是否清晰合理
- 武打/战斗场面是否有画面感和层次
- 江湖/修仙世界的氛围营造
- 侠义/道义主题的表达

### 都市现实
- 社会环境还原度
- 人物行为的现实可信度
- 日常细节的真实感
- 社会议题的处理深度

### 恐怖/惊悚
- 恐惧感的营造与持续
- 氛围与紧张感的层次递进
- 恐怖源的设定是否有效
- 心理压迫感的构建

### 历史
- 历史背景的准确性与还原度
- 虚构与史实的融合处理
- 时代语言与氛围
- 历史人物的塑造分寸

### 网文/爽文
- 爽点设计是否密集有效
- 节奏是否紧凑
- 金手指/系统设定的趣味性
- 读者代入感与翻页冲动

## 短篇 vs 长篇侧重差异

| 维度 | 短篇侧重 | 长篇侧重 |
|------|-----------|-----------|
| 情节架构 | 结构精巧，切入角度 | 多线交织，节奏控制 |
| 角色塑造 | 少数角色的深度 | 角色群像，关系网络 |
| 文笔风格 | 字句打磨，留白艺术 | 风格一致性，叙述耐力 |
| 世界观 | 暗示与留白 | 体系完整性 |
| 主题表达 | 精准聚焦 | 多层递进 |
| 情感共鸣 | 瞬间冲击 | 持续积累 |
| 创新性 | 构思巧妙 | 体系创新 |
| 可读性 | 一气呵成 | 长线吸引 |
