# 分类评测侧重

不同故事分类的核心卖点和阅读期待不同，评分权重应相应调整。

## 各分类专项要点

### 上古神话
- 忠实度权重最高之一：盘古、女娲、后羿等故事高度经典化，篡改容易被读者发现
- 氛围营造是难点：宏大叙事要靠缩小视角实现——写神的第一个感觉、第一个动作，不写全景航拍
- 角色与对话权重低：上古神话角色往往功能性强于个人性，对话少

### 四大民间传说
- 忠实度权重最高：白蛇传、梁祝等国民故事，读者有强预期，关键节点不可少
- 情感与余韵权重高：这四个故事的核心竞争力就是情感——爱情、离别、牺牲
- 篇幅通常为长篇：结构需要完整的四段式

### 仙凡奇缘
- 角色与对话权重高：仙凡相处的日常细节决定了故事有没有温度
- 情感与余韵权重高：离别处不煽情只放画面，是汪曾祺风格的核心考验
- 闲笔特别重要：日常相处中的食物、家务、闲话

### 聊斋志异
- 结构与节奏权重最高：聊斋原作结构精巧，重述需保持精密感
- 风格执行重要：聊斋涵盖明面（婴宁、小翠）和暗面（画皮、尸变），需准确切换
- "怪的东西写得越平常越有味道"是核心检验标准

### 志怪小说
- 各维度较均衡：志怪小说类型多样，不偏科
- 关注叙事手法的多样性（唐传奇有较强的故事性和人物性）

### 搜神志怪（魏晋六朝）
- 氛围营造权重高：魏晋志怪本身简练，一两个诡异细节放大即可，重在氛围
- 可读性权重高：短篇为主，要一口气读完
- 忠实度权重稍低：原作往往只有几十字，大量细节需要合理演绎

### 子不语·阅微（清代笔记怪谈）
- 氛围营造权重高：恐怖感靠具体感官（声音、气味、触感），不靠形容词
- 可读性权重最高：节奏偏快，读者期待一个短促有力的故事
- 忠实度权重最低：原作较短，重述空间大，合理演绎比逐字还原更重要
- 可适当加黑色幽默（纪晓岚原文就有）

### 民间恐怖与悬疑
- 氛围营造权重最高：氛围是这类故事的生命线——先铺日常再抽地面
- 可读性权重高：开头慢铺环境，但必须从第一句就留一个"不对劲"的钩子
- 忠实度权重最低：许多故事来自口传，版本多样，演绎自由度最大
- 结尾留悬念不必给解释

## 短篇 vs 中长篇侧重差异

| 维度 | 短篇侧重 | 中长篇侧重 |
|------|-----------|-----------|
| 忠实度 | 核心节点不漏即可 | 多版本处理、细节还原度 |
| 结构与节奏 | 结构精巧，两段/三段制的切割点 | 四段式的起承转合比例 |
| 风格执行 | 字句打磨，每个字都要站住 | 风格一致性和叙述耐力 |
| 氛围营造 | 一两个精准细节定调 | 多场景的氛围层次递进 |
| 角色与对话 | 少数角色的精准捕捉 | 角色关系和配角质量 |
| 情感与余韵 | 瞬间冲击，一个画面留住 | 持续积累，多个情感节点 |
| 可读性 | 一气呵成 | 长线吸引，章节钩子 |
