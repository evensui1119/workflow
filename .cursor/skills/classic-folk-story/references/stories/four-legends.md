# 四大民间传说

中国四大民间传说：牛郎织女、孟姜女哭长城、白蛇传、梁山伯与祝英台。

## 牛郎织女

- **出处**：《诗经·大东》（最早提及）、《荆楚岁时记》、各地口传
- **人物**：牛郎（牵牛）、织女、老牛、王母娘娘、喜鹊
- **梗概**：牛郎为孤儿，被兄嫂虐待赶出家门，仅得一头老牛。老牛通灵，指点牛郎趁织女等仙女下凡沐浴时取其天衣，织女遂留人间与牛郎成亲，生一儿一女。王母知晓大怒，遣天兵捉织女回天庭。老牛临死嘱牛郎披其皮可上天。牛郎携儿女追赶，王母拔簪划银河相隔。每年七月七日，喜鹊搭桥使二人相会。
- **关键细节**：老牛为金牛星下凡、织女为天帝孙女/王母外孙女、银河（天河）为分隔标志、七夕鹊桥相会
- **经典场景**：河边取衣、老牛托付、银河划开、鹊桥相会
- **主题**：忠贞爱情、人神之隔、反抗权威
- **相关节日**：七夕节（乞巧节）

## 孟姜女哭长城

- **出处**：最早可追溯至《左传》杞梁妻故事，后经历代演变
- **人物**：孟姜女、范喜良（万喜良）、秦始皇
- **梗概**：秦始皇修长城，范喜良被征为民夫。新婚之夜范喜良被抓走。孟姜女千里寻夫送寒衣，至长城下得知丈夫已死，尸骨筑入城墙。孟姜女痛哭三日三夜，长城为之崩塌八百里，露出丈夫骸骨。秦始皇闻知孟姜女美貌欲纳之，孟姜女提三条件（祭夫、修坟、游海），条件满足后投海而死。
- **关键细节**：滴血认骨（城墙崩后众骨混杂，孟姜女咬指滴血，血渗入骨者为夫骸骨）；部分版本有葫芦生人的奇异诞生情节
- **经典场景**：千里送寒衣、哭倒长城、滴血认骨、投海殉情
- **主题**：暴政之害、夫妻忠贞、反抗强权
- **变体**：各地有不同版本，范喜良也作万杞梁、范杞良等

## 白蛇传

- **出处**：明·冯梦龙《警世通言·白娘子永镇雷峰塔》（成型版本），后经弹词、戏曲不断丰富
- **人物**：白素贞（白蛇）、许仙（许宣）、小青（青蛇）、法海
- **梗概**：白蛇在峨眉山修炼千年化为美女白素贞，携青蛇小青下山。断桥上白素贞与许仙借伞相识，结为夫妻，在镇江开保和堂药铺。金山寺法海认出白蛇身份，设计端午让许仙劝白素贞饮雄黄酒现出原形，许仙惊死。白素贞盗仙草救活许仙。法海将许仙骗至金山寺，白素贞水漫金山救夫，因身怀六甲力竭。后生子许仕林。法海以金钵收白素贞镇于雷峰塔下。许仕林中状元祭塔救母（后世补叙）。
- **关键细节**：西湖借伞定情、游湖初遇、端午现原形、盗仙草（灵芝/蟠桃）、水漫金山、雷峰塔镇压
- **经典场景**：断桥借伞、端午惊变、昆仑盗草、水漫金山、合钵镇塔
- **主题**：人妖之恋、忠贞不渝、正邪之辩（法海究竟是正是邪？）
- **变体**：早期版本白蛇为恶蛇、许仙为贪色之人；后世逐渐美化为忠贞爱情故事

## 梁山伯与祝英台

- **出处**：东晋始有传说，唐·张读《宣室志》、宋代地方志多有记载
- **人物**：梁山伯、祝英台、马文才、师母/同窗
- **梗概**：祝英台女扮男装赴杭州（或他处）求学，与同窗梁山伯结为兄弟，同窗三载。祝英台暗生情愫，多次暗示梁山伯未解。十八相送时以种种比喻示爱，梁山伯仍未悟。后梁山伯得知英台为女子，赶往祝家求亲，但祝父已将英台许配马文才。梁山伯悲愤成疾而亡，官至县令而早逝（部分版本）。祝英台出嫁途中经过梁山伯坟前，下轿哭坟，坟墓裂开，英台纵身跳入，双双化为蝴蝶翩翩飞去。
- **关键细节**：女扮男装求学、三载同窗未辨雌雄、十八相送（最精彩桥段，各种隐喻如鸳鸯、鹅等）、楼台会（二人相认后再见）、哭坟化蝶
- **经典场景**：草桥结拜、三年同窗、十八相送、楼台相会、哭坟化蝶
- **主题**：纯真爱情、反抗封建婚姻、生死不渝
- **相关艺术**：小提琴协奏曲《梁祝》、越剧经典
