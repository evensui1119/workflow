# 东野圭吾风格卡（经典民间故事重述版）

本风格卡适用于**经典民间故事重述**场景下，用户要求「东野风格」「推理感」「真相反转」「像东野圭吾那样讲」时采用。冷静精密、信息控制、叙述性诡计、真相反转；重述经典时仍须保持人物、情节点、出处与参考文件一致，风格只作用于叙述方式、视角与节奏，不可为反转而篡改原故事结局与因果。

```
【风格卡：东野圭吾 / 经典民间故事重述】

叙述声音：
  人称：第三人称有限视角为主，也常用多视角轮转
  叙述者身份：冷静的旁观镜头，不介入评判
  距离：中距离——贴近角色行动和思维，但不过度内心独白
  可靠性：关键手法——叙述者给出的信息看似完整，
          实则精心省略了某个关键事实；
          读者以为自己看清了全貌，直到最后才发现被误导
  特色：视角切换本身就是叙事手段——
        谁的视角被呈现、谁的视角被隐藏，决定了谜面和谜底
  经典重述约束：原故事的人物、关键情节点、结局须与参考文件一致；
                可用双时间线、多视角、信息延迟披露来呈现同一故事，不可改写结局或因果

句法节奏：
  句长：短句和中句为主，干净利落
  关键技法：
    - 功能性散文：每一句都有信息量或推动作用，绝不虚写
    - 场景化推进：通过一个接一个的具体场景推进故事，
      而非大段的叙述概括
    - 对话密集：大量信息通过对话传递，对话即情节
    - 章节钩子：每章/每段结尾留一个悬念或新发现，
      驱动读者继续往下读
  段落：偏短，节奏明快，一页之内有多次信息跳转
  标点：句号频繁，段落感强；
        不用省略号制造廉价悬念，用信息的"戛然而止"制造真正的悬念

感官偏好：
  主导感官：视觉为主——文字像摄影机，记录动作和场景
  密度：克制，只在关键时刻铺开感官细节
  功能：感官不服务于氛围渲染，服务于线索埋设和心理暗示
  特色：
    - 日常细节的精确描写——不是为了好看，而是后面有用
    - 角色的微表情和小动作比风景描写更重要
    - "冷感官"——不追求热烈的感官冲击，
      用冷静的白描让读者自己感受到寒意

意象修辞：
  来源域：日常生活 / 社会关系
  密度：极低——几乎不用比喻，力量在于事实本身的冲击
  核心意象：
    - 不是意象驱动而是情节驱动——"那把刀"、"那封信"、"那通电话"
      这些具体物件成为叙事枢纽
    - 日常物品的"陌生化"：一个普通的东西因为真相揭露而变得触目惊心
  修辞风格：
    - 几乎不做修辞——如果一定要比喻，也是最朴素直白的
    - 力量来自"不修饰的真相"——赤裸裸地呈现事实，
      比任何修辞都有冲击力

时间处理：
  核心手法：双时间线或多时间线交织（在经典重述中可用于"考证"或"多版本"）
    - 现在的调查/追问 + 过去的传说/事件
    - 两条看似无关的线索最终交汇
  关键技法：
    - 时间跳跃不标注而是让读者自己发现
    - 同一事件在不同时间点呈现不同面貌
    - "时间差"本身就是谜题——什么时候发生的？真的是那个时候吗？
  在经典重述中的应用：
    - 用现代视角或旁观者视角切入古老传说 → 双时间线自然成立
    - 同一则经典在不同讲述者口中的差异 → 差异本身成为张力，但最终与参考文件的主流情节一致

对话风格：
  对话占比：高——大量信息通过对话传递
  特征：
    - 对话即推理：角色通过对话交换信息、试探、推断
    - 日常化：没有文学腔，角色说的都是正常人会说的话
    - 潜台词极重：表面在聊天，底下在博弈
    - 沉默比说话更有力——角色"没有回答"往往是最关键的信息
  在经典重述中的应用：
    - 村民/讲述者之间看似闲聊的对话暗藏线索
    - 知情人与追问者的对话像一场没有硝烟的审讯
    - 老人"不肯说"的部分比"说了"的部分更重要——但故事主线仍按参考文件推进

主题表达：
  核心主题群：
    - 人性的暗面——不是超自然的恐惧，是人心的恐惧
    - 爱与牺牲——为了保护某人可以做到什么程度？
    - 真相的代价——知道真相一定是好事吗？
    - 社会性的恶——不是个人的疯狂，是集体的沉默和纵容
  态度：不审判，只呈现；让读者自己感受道德的灰色地带
  结尾：真相揭露型——
    但真相往往不是"谁对谁错"那么简单，
    而是"为什么"的答案让人五味杂陈
  经典重述：原故事的结局与价值取向不改变，可在呈现方式上做"层层剥开"或"反转理解"

标志性技法：
  1. 叙述性诡计
     ——利用读者的阅读惯性制造误导
     你以为这个"他"是A，其实一直在写B；
     你以为这两件事同时发生，其实隔了十年
     在经典重述中：可用视角与信息控制让读者先误解、再与经典情节对照豁然开朗

  2. 结构反转
     ——故事的最后一段落彻底重构前面的认知
     不是简单的反转，而是让你重新理解前面的每一个场景
     在经典重述中：传说的"另一面"揭露后，
     你发现那个被当作反派的人其实有苦衷——但整体因果仍符合参考文件

  3. 冷酷的温情
     ——在极其冷静甚至残酷的叙述中，
     突然让你发现某个角色的深情或牺牲
     不煽情、不渲染，只是把事实摆出来——
     然后你自己被击中

  4. 日常的恐怖
     ——恐惧不来自超自然，来自人际关系的暗流
     邻居的微笑、亲人的沉默——
     这些日常表象下可能藏着深渊

禁忌（东野式经典重述绝不会做的事）：
  - 不篡改经典情节的结局与因果（重述须与参考文件一致，风格只改叙述方式）
  - 不用华丽的修辞和抒情段落
    （不要文学性的风景描写，除非那个风景是线索）
  - 不做全知叙述——信息控制是核心武器
    （读者只能知道作者允许他知道的）
  - 不在情感上做过度渲染
    （角色哭了就是"她哭了"，不要写"泪水像断了线的珠子"）
  - 不让超自然失去现实根基（若原故事含超自然）
    （即使写鬼，也要让读者始终觉得"说不定有另一个解释"）
  - 不写无功能的场景
    （每个场景都必须推进情节或埋设线索，否则删掉）
```

## 东野风格在经典重述中的适用场景

**最适合：**
- 有悬疑或冤情色彩的经典（如孟姜女、某些公案传说）——用信息控制与多视角层层揭开
- 同一故事多版本、多说法——用"考证"或"追问"结构呈现，最终落回主流版本
- 人性暗面突出的经典——冷静白描，不煽情地呈现善恶灰度

**需要调和：**
- 结局单一、情感浓烈的经典（如梁祝、牛郎织女）——可保留东野的节奏与对话密度，但不宜为反转而改写结局
- 纯神话/纯幻想类——东野风格倾向理性与信息控制，需与参考文件中的超自然设定相容

## 风格卡使用检查清单

每写完一个场景/段落，对照以下清单自检：

### 准确性与约束
- [ ] 人物、关键情节点、结局是否与参考文件一致？（未为风格而篡改）

### 信息控制
- [ ] 本节给出了什么信息？隐藏了什么信息？
- [ ] 读者此刻对真相的理解是否正好是我想让他们以为的？
- [ ] 有没有无意中泄露了不该泄露的线索？

### 叙述效率
- [ ] 每个场景是否都有推进情节或埋设线索的功能？
- [ ] 有没有纯粹的氛围段落可以压缩？
- [ ] 对话是否在传递信息或揭示角色？

### 反转准备（若采用反转结构）
- [ ] 最终反转所需的所有伏笔是否已埋好？
- [ ] 伏笔是否自然到读者第一遍读不会注意？
- [ ] 反转后回看，前面的场景是否都有新的含义？且与参考文件一致？

### 禁忌检查
- [ ] 有无过度抒情的段落？
- [ ] 有无无功能的风景描写？
- [ ] 有无全知视角的泄露？
- [ ] 情感是否被过度渲染？
