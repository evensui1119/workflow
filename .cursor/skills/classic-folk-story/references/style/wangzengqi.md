# 汪曾祺风格卡（经典民间故事重述版）

本风格卡适用于**经典民间故事重述**场景下的默认风格。语言干净松弛，细节精准，带一点温润的幽默；像白描水墨画，不抢故事的风头。重述经典时仍须保持人物、情节点、出处与参考文件一致，风格只作用于叙述方式、感官与节奏，不可为风格而篡改情节。

```
【风格卡：汪曾祺 / 经典民间故事重述】

叙述声音：
  人称：第三人称为主，偶尔用"我们那边""据说"等带出讲述者存在感
  叙述者身份：隐在故事后面的人，不议论不评判，
            但偶尔冒出来补一句闲话，像邻桌有人在聊天
  距离：中近距离——贴着人物的日常生活写，
        不居高临下俯瞰，也不钻进人物脑子里做大段独白
  可靠性：不声称全知，遇到说不清的地方会用
          "大概是""谁知道呢""后来的事没人说得清"
  温度：温的。不冷，不热，像春天的太阳
  经典重述约束：人物名、关键情节点、出处严格依参考文件，
              风格只改叙述方式与感官质地，不改情节与因果

句法节奏：
  句长：短句和中句为主，偶尔来一个稍长的句子拉出节奏感
  关键技法：
    - 松弛感：句子之间留气口，不赶。
      读起来像散步，不像跑步
    - 闲笔：会花几句话写一棵树、一碗饭、一个人走路的样子，
      看似跑题，其实在造氛围、建质感
    - 短句收束：一段写完，常用一个很短的句子兜住
      "就这样。""后来就好了。""谁也没再提。"
    - 不排比：几乎不用排比句。
      一个意思说一遍就够了，不需要说三遍来加强
  段落：偏短，一段一件事或一个画面
  标点：句号多于逗号。每个句子是独立的、站得住的

感官偏好：
  主导感官：视觉为主——但不是电影镜头，是水墨画
  密度：适中——不堆砌，但关键处一定有
  功能：让场景可信，让人物有体温，让故事落在实处
  特色：
    - 视觉偏淡：不写"血红""漆黑"这类浓烈色彩，
      写"天有一点灰""水是青的""她穿一件月白的褂子"
    - 食物是招牌：汪曾祺笔下永远有吃的。
      一碗粥怎么熬的、一条鱼怎么做的、
      集市上卖什么点心——这些细节让故事有了烟火气
    - 声音偏安静：虫叫、水响、风过树梢、远处有人喊了一声
    - 气味偶尔出现：花香、饭香、雨后泥土的味道。
      不写腐烂、血腥——那不是他的路子
    - 触觉克制：偶尔写一下——
      "手伸进水里，凉的""太阳晒在背上，暖洋洋的"

意象修辞：
  来源域：自然 / 日常生活 / 食物 / 水
  密度：低——比喻很少，有也极朴素
  核心手法：
    - 不说"像"，用具体细节替代比喻。
      不说"她好看得像仙女"，写"她挎着篮子从桥上过，
      河里的水都亮了一下"
    - 不用"不是……是……"句式
    - 风景即情感：人高兴时写天晴，人难过时写下雨，
      但不点破，让读者自己对上
    - 白描为主：一个准确的动词、一个恰当的名词，
      比三个形容词有用
  修辞风格：
    - 克制到近乎吝啬。好句子不是写出来的，是省出来的
    - 偶尔来一句有味道的话，像菜里放的那一点点盐

时间处理：
  基本线性，不玩结构
  关键技法：
    - 日常时间：用节气、农事、民俗标记时间
      "快到端午了""麦子该收了""腊月二十三，灶王爷上天"
    - 慢处极慢：一顿饭可以写半页，一个下午可以写一整段
    - 快处极快："过了两年""后来他就走了"一笔带过
    - 不标注"几年后""此时此刻"——
      时间在细节里自然流过，不需要路标

对话风格：
  对话占比：适中——不像东野那么密集，也不像莫言那么少
  特征：
    - 自然到像偷听来的：人说人话，不说书面语
    - 简短：农民不会说长句子。
      "行。""不行。""你看着办。""吃了没？"
    - 有性格：不同人说话不一样。
      老人慢，年轻人快，女人细，汉子粗。
      但都是几个字就把人物立住
    - 对话标签极简："说""道""问"，偶尔"笑了笑说"
    - 沉默也是对话：有时一个人说完，另一个人没接话。
      不写"他沉默了"，就是不接话。留白
  古典题材处理：略带古意但不拗口。
    不说"小生有礼了"，说"他拱了拱手"

主题表达：
  方式：完全隐性——故事讲完就完了，不点题
  态度：温厚。不批判，不说教，不替人物做道德选择
    对好人有善意，对坏人也不咬牙切齿
    对苦难不渲染——"日子就是这样过的"
  结尾：常以一个画面或一个细节收束
    "院子里的栀子花开了。"
    "他走了。街上卖糖人的还在吆喝。"
    不总结，不升华，不"这个故事告诉我们"
  经典重述：不改变原故事的因果与价值取向，
          不加入原故事没有的深刻寓意

标志性技法：
  1. 闲笔
     ——看似无关的细节，其实在建造整个世界
     写一个人去赶集，先写集市上有什么：
     卖豆腐的、卖布的、耍猴的、算命的。
     这些跟故事主线无关，但读完你觉得你去过那个集市

  2. 食物描写
     ——汪曾祺的故事里永远有吃的
     不是为了写吃，是为了写人。
     一个人怎么吃饭、吃什么、跟谁吃，
     比任何心理描写都管用

  3. 风景收束
     ——用一个自然景物的画面结束一段或一篇
     不评论，不抒情，就放一个画面在那里
     读者看见那个画面，自己会有感觉

  4. 温润的幽默
     ——不是抖机灵，不是讽刺，是一种善意的好笑
     写一个人的小毛病、小习惯，让人会心一笑
     笑完了又觉得这个人可亲

  5. 白描立人
     ——不写"他是一个善良的人"
     写他做了什么、说了什么、穿什么、吃什么。
     三五个细节，人就立在那里了

禁忌（汪曾祺式经典重述绝不会做的事）：
  - 不篡改经典情节、人物与出处（重述须与参考文件一致）
  - 不抒情——绝不写"啊，多么美丽的月光"
    （要写月光，就写"月亮出来了，地上白白的"）
  - 不说教——绝不在结尾总结道理
  - 不堆砌形容词——一个够用绝不用两个
  - 不排比——说一遍就够了
  - 不用华丽的比喻——"眼如秋水"这种东西不会出现
  - 不写大段内心独白——人物的心思通过动作和细节传达
  - 不渲染苦难——苦就是苦，不需要加感叹号
  - 不急——节奏永远是松弛的，哪怕写打仗也不慌
```

## 汪曾祺风格在经典重述中的适用场景

**最适合：**
- 有烟火气和日常生活的故事（田螺姑娘、天仙配、牛郎织女）
- 有人情味的故事（民间智者、寓言警世）
- 需要氛围而非紧张感的故事（仙凡奇缘、部分聊斋）
- 爱情故事——汪曾祺写感情是淡的、暖的，适合民间传说的基调

**需要调和：**
- 恐怖/惊悚类（画皮、尸变）——汪曾祺的温度太暖，需要适度收冷
- 需要强烈冲突的故事（荆轲刺秦、霸王别姬）——汪曾祺的松弛节奏可能托不住激烈场面，高潮处需适度加快
- 上古神话的宏大叙事（盘古开天、女娲补天）——汪曾祺擅长小场景，大场面需要适度放开

## 风格卡使用检查清单

每写完一个场景/段落，对照以下清单自检：

### 准确性与约束
- [ ] 人物、关键情节点、出处是否与参考文件一致？

### 语言质地
- [ ] 句子是否干净？能删的字删了没有？
- [ ] 有没有排比？有的话删掉
- [ ] 有没有"像……"的明喻？能换成具体细节吗？
- [ ] 有没有形容词堆砌？一个够用的地方是不是用了三个？
- [ ] 读起来是不是松弛的？有没有在赶？

### 感官与细节
- [ ] 有没有至少一处食物或日常生活的细节？
- [ ] 感官是不是偏淡、偏暖的？有没有过于浓烈的色彩或气味？
- [ ] 闲笔有没有？（看似无关但建造世界的细节）

### 人物与对话
- [ ] 对话是不是简短、自然、有性格？
- [ ] 人物是通过细节立住的，还是通过形容词贴标签？
- [ ] 有没有留白？（该沉默的地方沉默了没有）

### 结尾与主题
- [ ] 结尾是不是一个画面或一个细节？
- [ ] 有没有跳出来点题或说教？
- [ ] 有没有"这个故事告诉我们……"？有的话删掉

### 禁忌检查
- [ ] 有无抒情段落？
- [ ] 有无大段内心独白？
- [ ] 有无渲染苦难？
- [ ] 节奏是否始终松弛？
