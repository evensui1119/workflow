# 白话演绎风格卡（经典民间故事重述）

本风格卡专用于「中国经典民间故事」的讲述与重述：现代白话、叙述自然，注重生动细腻的描写与心理刻画，不采用说书人体。若用户明确要求说书人体，则改用说书人腔调与结构，并另行参考说书人风格卡。

```
【风格卡：白话演绎 / 经典民间故事重述】

叙述声音：
  人称：第三人称全知或限知
  框架：直接讲故事，无「列位看官」「且听我道来」等说书人套话
  叙述者身份：隐身的讲述者，不插嘴议论、不跳出来点评
  距离：可近可远——铺垫时略远、高潮时拉近，用心理与感官贴紧人物
  可靠性：重述经典时严格依参考文件，人物、情节点、出处不篡改；
          多版本时说明主流版本，可简要提及变体

句法节奏：
  句长：以中短句为主，可随情绪变化——紧张处短句快推，抒情处放缓拉长
  关键技法：
    - 节奏有张弛：紧张处短句、快动作；抒情处句子拉长、细节铺陈
    - 避免连续排比铺陈，不滑向朗诵腔
    - 段落长度随场景变化，高潮前可放慢、高潮时加快
  标点：自然多样，不过度使用省略号或感叹号

感官与细节：
  主导感官：五感并用，让读者「看见」场景——颜色、声音、气味、温度
  密度：关键场景需有具体感官细节，建立可信感
  功能：用生活细节与感官让经典情节「活」在当下阅读体验中
  特色：
    - 视觉：场景、服饰、神态、光影
    - 听觉：对话、环境声、沉默
    - 触觉与温度：冷热、粗粝与柔软、风雨
    - 嗅觉与味觉：在饮食、环境、情绪节点适当出现

心理与情感：
  要求：适当通过心理描写与情感细节增强代入感
  方式：内心活动自然融入叙述，不堆砌「他想」「她感到」
  人物：给角色符合身份的语言、动作、小习惯，让人物活起来

修辞与语言：
  基调：现代口语、叙述自然，不文绉绉也不刻意粗俗
  比喻：可用但克制，优先具象细节；少用「像……」「不是……是……」的套路句式
  禁忌：拒绝词藻堆砌；一个准确的动词胜过三个华丽的形容词
  民间故事力量：在于「说事儿」和情感真，而非炫文采

时间与结构：
  结构：铺垫 → 发展 → 高潮 → 结局
    铺垫：交代人物、时代、地点，用生活细节建立可信感
    发展：核心情节展开，每个转折处加强节奏感
    高潮：故事最精彩处，放慢叙述、加强感官细节
    结局：交代结果，留有余韵
  时间：线性叙述为主，可适度压缩「过了几年」等过渡

对话风格：
  对话占比：与叙述平衡，关键情节点用对话推进
  特征：符合人物身份与时代感，不现代化过头
  标签：简洁自然，「说」「道」「问」等，不堆砌复杂标签
  古典题材：可略带古意或含蓄，但整体保持可读的白话

主题与结尾：
  方式：通过故事本身呈现，不跳出来点题
  结尾：交代结果，留有余韵；可留白，不必解释过度
  经典重述：不改变原故事的价值取向与因果，不擅自「升华」或道德说教

标志性技法：
  1. 准确性第一
     人物名字、关系、出处严格按参考文件；关键情节点不遗漏不篡改
     （如「十八相送」之于梁祝、「滴血认骨」之于孟姜女）

  2. 感官细节落地
     让读者看见、听见、触到场景，用细节建立可信感与代入感

  3. 人物活起来
     符合身份的语言、动作、小习惯；心理与情感自然融入叙述

  4. 节奏有张弛
     紧张处短句快推，抒情处放缓拉长；高潮前可放慢，高潮时加快

  5. 无说书人套话
     不采用「列位看官」「且听我道来」及说书人插嘴议论（除非用户明确要求说书人体）

禁忌（白话演绎经典重述绝不会做的事）：
  - 不采用说书人体套话与插嘴议论（除非用户明确要求）
  - 不张冠李戴、不遗漏或篡改关键情节点与出处
  - 不在结尾强行道德说教或「这个故事告诉我们……」
  - 不堆砌形容词、不连续排比铺陈、不书面腔
  - 不把经典故事现代化到违背时代感（如古人说网络用语）
```

## 风格卡使用检查清单

讲述或修订时，对照以下清单自检：

### 叙述声音
- [ ] 是否在直接讲故事，无说书人套话与插嘴？
- [ ] 叙述者是否隐身、不跳出来点评？

### 准确性与结构
- [ ] 人物、情节点、出处是否与参考文件一致？
- [ ] 铺垫 → 发展 → 高潮 → 结局是否清晰？
- [ ] 结尾是否留有余韵、未过度解释？

### 感官与人物
- [ ] 关键场景是否有感官细节（至少两种感官）？
- [ ] 人物是否有符合身份的语言、动作或小习惯？
- [ ] 心理与情感是否自然融入、未堆砌？

### 语言与节奏
- [ ] 句长与节奏是否随情绪有张弛？
- [ ] 有无「像……」「不是……是……」或排比泛滥？
- [ ] 有无词藻堆砌、书面腔？

### 禁忌检查
- [ ] 是否未使用说书人套话（除非用户要求）？
- [ ] 是否未篡改经典情节与出处？
- [ ] 是否未在结尾强行点题或说教？
